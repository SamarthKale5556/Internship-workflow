## 🚀 Overview

Blind Assist AI is a lightweight assistive navigation system designed to help visually impaired users move safely without relying on continuous internet connectivity.

The system performs real-time object detection, obstacle awareness, and audio-based warning alerts directly on-device using Edge AI, making it suitable for low-cost wearable solutions.

---

## ✨ Current Features

✅ Real-Time Object Detection

✅ Offline Operation (No Internet Required)

✅ Low-Latency Audio Alerts

✅ Person, Vehicle & Obstacle Awareness

✅ Bluetooth Audio Support

✅ ONNX Optimized Model

✅ Raspberry Pi Ready
